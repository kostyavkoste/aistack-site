./hermes_memory

Шаг 2 · Память команды (Hermes)

Hermes — это «память» вашей команды. Без него агенты забывают контекст между разговорами. С ним — Агент.exe помнит ваших клиентов, проекты, прошлые решения неделями и месяцами.

🧠
Что это значит на практике
Без Hermes: каждое утро объясняете команде «у меня клиент Иван, ниша салон красоты, запуск 1 июня».
С Hermes: команда сама помнит. Спросили «как там Иван?» — отвечают без напоминаний.
✓ Да, установить автоматически REC
setup-скрипт сам всё поставит на ваш компьютер. Ничего делать не надо.
⚙ У меня уже стоит
только для тех, кто сам поставил Hermes заранее
Что делает setup-скрипт: ставит Hermes в ~/.openclaw/hermes/, запускает его как сервис на порту 7777, подключает к OpenClaw как memory-provider. На macOS через LaunchAgent, на Windows через Scheduled Task, на Linux через systemd. Команды и проверки — внутри bundle.zip, вы их не пишете руками.
